# Converting fonts

- [LVGL: Font Converter](https://lvgl.io/tools/fontconverter)
    - Settings:
        - Name: default
        - Size:
            - 20
            - ...(?)
        - Bpp: 8 bit
        - No fallback
        - Output format: Binary
        - DO not check: font compression, horizontal subpixel rendering and glyph color.
